# React Card App

A simple React application built with **Vite**. This project demonstrates reusable React components, CSS styling, and a responsive card layout.

## Features

* Built with React and Vite
* Reusable Card component
* Responsive design
* Clean and simple UI
* Fast development with Vite

## Tech Stack

* React
* Vite
* JavaScript (ES6+)
* CSS3
* HTML5

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### Navigate to the project

```bash
cd your-repository-name
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
